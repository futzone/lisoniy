## Umumiy Qoidalar

Qaysi sohada ishlashingizdan qat'i nazar:

### 1. Rol bering
```
"Sen tajribali [kasb] sifatida..."
```

### 2. Kontekst bering
```
"Auditoriya: [kim]
Maqsad: [nima]
Cheklov: [qanday]"
```

### 3. Tekshiring
```
SI javobini har doim tekshiring!
Ayniqsa faktlar va raqamlar.
```

### 4. Iteratsiya qiling
```
Birinchi javob mukammal bo'lmasa — qayta so'rang.
"Buni qisqartir" / "Batafsilroq" / "Boshqa variant"
```
