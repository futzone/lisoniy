# 3.1 Ta'lim va O'qituvchilar Uchun

## SI — O'qituvchining Eng Yaxshi Yordamchisi

O'qituvchilar uchun SI vaqtni sezilarli tejashga yordam beradi: dars rejalar, testlar, individual yondashuv.

---

## Asosiy Foydalanish Yo'llari

### 1. Dars Rejasi Yaratish

```
Sen tajribali [fan] o'qituvchisisan.

[Mavzu] bo'yicha 45 daqiqalik dars rejasi yoz.

O'quvchilar: [sinf], [daraja]

Struktura:
- Kirish (5 daq): E'tibor jalb qilish
- Asosiy qism (30 daq): Tushuntirish + amaliyot
- Mustahkamlash (10 daq): Savol-javob

Format: Jadval ko'rinishida, vaqt belgilangan
```

### 2. Test Yaratish

```
[Mavzu] bo'yicha test yarat.

Parametrlar:
- 10 ta savol
- 4 ta variant (A, B, C, D)
- Qiyinlik: o'rta
- Javoblar alohida sahifada

Format: Raqamlangan, aniq
```

### 3. Individual Tushuntirish

```
Sen sabr-toqatli o'qituvchisan.

[Mavzu]ni [sinf] o'quvchisiga shunday tushuntirilki:
- Oddiy so'zlar
- Hayotiy misollar
- Qadam-baqadam
- Rasm/diagramma tavsifi

Agar o'quvchi tushunmasa — boshqa usulda qayta tushuntir.
```

---

## Tayyor Promptlar

### Dars rejasi:
```
Matematika, 7-sinf, "Kvadrat tenglama" mavzusi.
45 daqiqalik dars rejasi. Interaktiv mashqlar qo'sh.
```

### Test:
```
Biologiya, 9-sinf, "Hujayra tuzilishi".
15 ta test savoli, 4 variant, javoblar bilan.
```

### Tushuntirish:
```
5-sinf o'quvchisiga kasrlarni qo'shishni
pizza misolida tushuntir.
```

### Baholash:
```
O'quvchi inshosini ko'rib chiq:
[Insho matni]

Baholash mezonlari:
- Grammatika (25%)
- Mazmun (25%)
- Struktura (25%)
- Kreativlik (25%)

Batafsil fikr-mulohaza yoz.
```

## Amaliy Misollar

### Differentiated Learning (Individual yondashuv)

```
Bir xil mavzuni 3 xil darajada tushuntir:

Mavzu: Fotosintez

1. BOSHLANG'ICH (5-sinf):
   Oddiy so'zlar, rasm bilan

2. O'RTA (8-sinf):
   Ilmiy terminlar, formula

3. YUQORI (11-sinf):
   Molekulyar daraja, Calvin sikli
```

### Flipped Classroom

```
"Pythagor teoremasi" mavzusi uchun:

1. Uy vazifasi sifatida 5 daqiqalik video skripti
2. Sinfda muhokama uchun 5 ta savol
3. Guruh loyihasi topshirig'i
```
