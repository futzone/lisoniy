# 3.5 Moliya va Buxgalteriya

## SI Moliyada — Tahlil va Hisobot Yordamchisi

SI moliya sohasida **tahlil va hisobotlar** tayyorlashda yordam beradi. Lekin muhim qarorlar uchun mutaxassis kerak.

---

## Asosiy Foydalanish

### 1. Moliyaviy Hisobotlar

```
Quyidagi ma'lumotlar asosida [hisobot turi] tayyorla:

Ma'lumotlar:
[Daromad, xarajat, foyda, etc.]

Format: [Jadval / Matn / Taqdimot]
Auditoriya: [Direktor / Investor / Jamoa]
Davr: [Oy / Chorak / Yil]
```

### 2. Byudjet Rejalashtirish

```
[Kompaniya/Loyiha] uchun [davr] byudjeti:

Parametrlar:
- Umumiy byudjet: [summa]
- Bo'limlar: [ro'yxat]
- Ustuvorliklar: [ro'yxat]
- Kutilmagan xarajat fondi: [%]

Format: Jadval + tushuntirish
```

### 3. Moliyaviy Tahlil

```
Quyidagi ma'lumotlarni tahlil qil:

[Ma'lumotlar]

Tahlil turi: [Trend / Comparative / Ratio]
Xulosa: Kuchli va zaif tomonlar
Tavsiya: 3-5 ta aniq qadam
```

---

## Tayyor Promptlar

### Oy hisoboti:
```
2024-yil Yanvar oyi uchun moliyaviy hisobot:
Daromad: 150 mln
Xarajat: 120 mln
Kategoriyalar: [...]
Oldingi oy bilan taqqoslash bilan.
```

### Startup byudjet:
```
MVP ishlab chiqish uchun 6 oylik byudjet:
Umumiy: $50,000
Kategoriyalar: Development, Design, Marketing, Operations
Oylik breakdown bilan.
```

### ROI hisoblash:
```
Marketing kampaniyasi ROI:
Xarajat: $5,000
Natija: 200 ta yangi mijoz
O'rtacha buyurtma: $50
LTV: $200

ROI va payback period hisobla.
```

---

## Murakkab Tahlil

### Break-Even Analysis

```
Break-even nuqtasini hisobla:

Fixed costs: [...]
Variable cost per unit: [...]
Selling price per unit: [...]

Formula bilan, grafik tavsifi bilan.
```

### Cash Flow Projection

```
6 oylik cash flow prognozi:

Dastlabki balans: [...]
Oylik daromad (prognoz): [...]
Oylik xarajatlar (prognoz): [...]
Seasonality: [...]

Jadval + risk ssenariylari.
```

---

## ⚠️ Muhim Cheklovlar

SI **qila olmaydi**:
- ❌ Buxgalteriya hisobi yuritish
- ❌ Soliq deklaratsiyasi to'ldirish
- ❌ Audit o'tkazish
- ❌ Moliyaviy maslahat berish (professional ma'noda)

SI **yordam beradi**:
- ✅ Ma'lumotlarni formatlash
- ✅ Hisob-kitob tekshirish
- ✅ Hisobot shablonlari
- ✅ Tahlil va vizualizatsiya tavsifı

---

## Xulosa

SI moliyada **yordamchi vosita**:
- Vaqtni tejaydi
- Hisobotlar tayyorlaydi
- Tahlil qilishga yordam beradi

**Lekin:** Muhim qarorlar uchun buxgalter yoki moliyaviy maslahatchi bilan maslahatlashing.
