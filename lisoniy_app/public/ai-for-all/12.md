## Ruxsat Etilgan Foydalanish

### 1. Hujjat Shablonlari

```
O'zbekiston qonunchiligiga mos 
[hujjat turi] shabloni yoz.

Misol: Ish shartnomasi, Ijara shartnomasi, NDA

Format: To'liq struktura, bo'sh joylar [___] bilan
Til: Rasmiy, huquqiy

Eslatma: "Huquqshunos bilan ko'rib chiqing" qo'sh.
```

### 2. Qonun Tahlili

```
[Qonun/Modda] ning asosiy nuqtalarini oddiy tilda tushuntir.

Auditoriya: Huquqshunos emas, oddiy fuqaro
Format: Ro'yxat, misollari bilan

Manba: O'zbekiston qonunchiligiga murojaat
```

### 3. Shartnoma Tekshiruvi

```
Quyidagi shartnomani ko'rib chiq:

[Shartnoma matni]

Tekshir:
1. Noaniq bandlar
2. Potensial risklar
3. Yo'qolgan muhim bandlar
4. Tavsiyalar

Format: Jadval + tushuntirish
```

## Tayyor Promptlar

### Ish shartnomasi:
```
O'zbekiston Mehnat Kodeksiga mos 
IT kompaniya uchun ish shartnomasi shabloni.
Sinov muddati, NDA, intellectual property bandlari bilan.
```

### Ijara shartnomasi:
```
Kvartira ijarasi shartnomasi.
Bandlar: Muddat, to'lov, kafolat, javobgarlik.
Ikkala tomon uchun himoya.
```

### Huquqiy xat:
```
Mijozga yetkazilmagan xizmat uchun 
rasmiy da'vo xati shabloni.
Professional, lekin qat'iy ohangda.
```

## Amaliy Misol: Shartnoma Tahlili

```
Quyidagi ish shartnomasi bandini tahlil qil:

"Xodim kompaniyadan ketgandan so'ng 2 yil davomida 
raqobatchi kompaniyada ishlay olmaydi."

Tahlil:
1. Bu qonuniymi? (O'zbekiston qonunchiligida)
2. Ijro etilishi mumkinmi?
3. Xodim uchun risk
4. Tavsiya

[SI javobi...]

⚠️ Bu umumiy ma'lumot. Aniq holatda huquqshunos bilan maslahatlashing.
```
