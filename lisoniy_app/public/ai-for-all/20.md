# 4.5 Jamoaviy Ish va Hamkorlik

## SI — Jamoa Samaradorligini Oshiruvchi

Jamoa bilan ishlashda SI kommunikatsiya va koordinatsiyani yaxshilaydi.

---

## Uchrashuv Yordami

### Meeting Prep

```
Ertangi uchrashuv uchun tayyorgarlik:

Mavzu: [...]
Ishtirokchilar: [...]
Oldingi uchrashuv qarorlari: [...]
Hozirgi muammolar: [...]

Tayyorla:
1. Agenda (10 punkt max)
2. Har bir punkt uchun vaqt
3. Muhokama uchun savollar
4. Kerakli materiallar ro'yxati
```

### Real-time Notes

```
Uchrashuv transkripsiyasidan notes yarat:

[Transkript]

Format:
- Key decisions (qarorlar)
- Action items (kim, nima, qachon)
- Parking lot (keyinga qoldirilgan)
- Follow-up kerak bo'lgan mavzular
```

### Post-Meeting Summary

```
Uchrashuv yakunlandi. Xulosa yubor:

Ishtirokchilarga:
- 5 jumlada xulosa
- Action items ro'yxati
- Keyingi uchrashuv

Ishtirok etmaganlarga:
- 3 jumlada xulosa
- Ular uchun relevant ma'lumot
```

## Task Management

### Task Description Generator

```
Quyidagi g'oya asosida task description yoz:

G'oya: [...]
Loyiha: [...]
Kim bajaradi: [junior/mid/senior]

Include:
- Aniq maqsad
- Acceptance criteria (3-5 ta)
- Kerakli resurslar
- Estimated time
- Dependencies
```

### Sprint Planning Assistant

```
Quyidagi backlog dan sprint planning qil:

Backlog items: [...]
Team capacity: [developer × hours]
Sprint length: [hafta]

Output:
1. Selected items (priority bo'yicha)
2. Assignments
3. Risk assessment
4. Buffer recommendation
```

## Kommunikatsiya Yordami

### Slack/Teams Message Drafter

```
Quyidagi vaziyat uchun jamoa xabari yoz:

Vaziyat: [...]
Auditoriya: [kimlar]
Maqsad: [inform/request/announce]
Urgency: [low/medium/high]

Ohang: Professional lekin do'stona
Uzunlik: 3-5 jumla
Include: emoji (agar appropriate)
```

### Conflict Resolution Helper

```
Quyidagi vaziyatda yordam ber:

Muammo: [ikki jamoa a'zosi o'rtasida kelishmovchilik]

Vazifa:
1. Har ikki tomonning pozitsiyasini neutral tilda bayonla
2. Umumiy maqsadni aniqla
3. Win-win yechim taklif qil
4. Keyingi qadam tavsiya qil

Ohang: Neytral, empathetic
```

## Documentation Collaboration

### Shared Doc Review

```
Jamoa hujjatini ko'rib chiq:

[Hujjat]

Tekshir:
1. Izchillik (terminology, format)
2. Gaps (nima yetishmayapti)
3. Clarity (tushunarlilik)
4. Suggestions (yaxshilash)

Format: Inline comments uslubida
```

### Knowledge Base Article

```
Jamoa uchun [mavzu] bo'yicha KB article yoz:

Auditoriya: [yangi/barcha jamoa a'zolari]
Technical level: [beginner/intermediate/advanced]

Structure:
1. Overview (5 jumla)
2. Step-by-step guide
3. FAQ (5 ta)
4. Related articles (taklif)
```
