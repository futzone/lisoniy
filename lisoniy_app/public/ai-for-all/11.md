# 3.2 Tibbiyot va Sog'liqni Saqlash

## ⚠️ MUHIM OGOHLANTIRISH

**SI shifokor EMAS va HECH QACHON:**
- Diagnostika qo'ymaydi
- Dori-darmon tavsiya etmaydi
- Shifokor o'rnini bosmaydi

SI faqat **yordamchi vosita** — tadqiqot, hujjatlar va tushuntirish uchun.

---

## Ruxsat Etilgan Foydalanish

### 1. Tibbiy Tadqiqot Yordami

```
Sen tibbiy tadqiqotchi sifatida yordam ber.

[Kasallik] bo'yicha so'nggi 5 yildagi 
asosiy tadqiqotlar haqida xulosa yoz.

Manba: PubMed, peer-reviewed jurnallar
Format: Ro'yxat, qisqa tavsif bilan
```

### 2. Bemor Uchun Tushuntirish

```
Sen sabr-toqatli shifokorsan.

[Kasallik/Jarayon]ni bemorga oddiy tilda tushuntir:
- Tibbiy terminlarsiz
- Hayotiy misollar bilan
- Qo'rquvsiz, tinchlantirib

Muhim: "Shifokoringiz bilan maslahatlashing" deb qo'sh.
```

### 3. Tibbiy Hujjatlar

```
Quyidagi bemor ma'lumotlarini SOAP formatida yoz:

Bemor shikoyati: [...]
Tekshiruv: [...]
Tahlillar: [...]

Format: SOAP (Subjective, Objective, Assessment, Plan)
```

---

## Tayyor Promptlar

### Tibbiy terminlarni oddiy tilda:
```
"Gipertenziya" ni bemorga tushuntiradigan
3 jumlada yoz. Qo'rqitmasdan, faktlar bilan.
```

### Tadqiqot xulosa:
```
2-toifa diabetni boshqarishda 
metforminning samaradorligi haqida qisqa xulosa.
Manba: So'nggi meta-tahlillar.
```

### Profilaktika maslahati:
```
Ofis xodimi uchun orqa og'rig'ini oldini olish 
bo'yicha 5 ta amaliy maslahat.
Shifokor konsultatsiyasi tavsiyasi bilan.
```
