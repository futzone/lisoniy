# 5.1 Maxfiy Ma'lumotlar va Xavfsizlik

## SI ga Nima Yuborish Mumkin?

SI ga yozgan har bir so'zingiz **serverga yuboriladi**. Bu nimani anglatadi?

---

## Asosiy Muammo

```
Siz yozdingiz: "Kompaniyamiz 2025-yil Q1 da $5M zarar ko'rdi..."
     ↓
SI serveriga yuborildi
     ↓
Potensial saqlandi / o'rganish uchun ishlatildi
     ↓
❌ Maxfiy ma'lumot xavf ostida
```

---

## ❌ HECH QACHON Yubormang:

### 1. Shaxsiy Ma'lumotlar
- Pasport raqami
- Bank karta raqamlari
- Parollar
- Manzil, telefon (boshqalarning)
- Tibbiy ma'lumotlar

### 2. Kompaniya Sirlari
- Moliyaviy hisobotlar (chiqarilmagan)
- Mijozlar ro'yxati
- Narxlash strategiyasi
- Texnik sirlar
- Xodimlar shaxsiy ma'lumotlari

### 3. Huquqiy Hujjatlar
- NDA ostidagi ma'lumotlar
- Sud materiallari
- Tekshiruv hujjatlari

---

## ✅ Yuborish Mumkin:

### 1. Umumiy Ma'lumot
```
✅ "Marketing strategiyasi qanday yoziladi?"
✅ "Python da list qanday ishlatiladi?"
✅ "Email shabloni yoz"
```

### 2. Anonimizatsiya Qilingan
```
❌ "Ahmedov Karim uchun shartnoma yoz"
✅ "[MIJOZ_ISMI] uchun shartnoma shabloni yoz"

❌ "Kompaniyamiz 5 mln dollar zarar ko'rdi"
✅ "Kompaniya zarar ko'rganda qanday hisobot yoziladi?"
```

### 3. Ochiq Ma'lumot
```
✅ Umumiy bilim
✅ O'zingiz yozgan kontent (copyright sizda)
✅ Ochiq manbalardan
```

## Xavfsiz Foydalanish Qoidalari

### Qoida 1: "Gazeta testi"
```
"Bu ma'lumot gazetada chiqsa — muammomi?"
Ha → Yubormang
Yo'q → Yuborishingiz mumkin
```

### Qoida 2: Anonimlashtiring
```
Oldin: "Toshkent shahridagi Karimov Anvar, +998901234567..."
Keyin: "[SHAHAR]dagi [MIJOZ], [TELEFON]..."
```

### Qoida 3: Local Alternativalar
Juda maxfiy ma'lumot uchun — local modellar:
- Llama 3 (kompyuteringizda)
- Ollama
- LM Studio

## Amaliy Mashq

### Quyidagilarni baholang:

   - **Javob:** ✅ OK — umumiy savol
