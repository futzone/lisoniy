# 2-Modul: Prompt Engineering Asoslari

## SI Bilan Samarali Muloqot Qilish San'ati

**Prompt** — bu siz SIga beradigan ko'rsatma yoki savol. Prompt sifati = Javob sifati.

---

## Nega Prompt Engineering Muhim?

Bir xil savolni ikki xil tarzda berishingiz mumkin:

### ❌ Yomon prompt:
```
"Marketing haqida yoz"
```
*Natija: Umumiy, foydasz matn*

### ✅ Yaxshi prompt:
```
"Sen tajribali marketing menejerisin. 
O'zbekistondagi kichik bizneslar uchun Instagram marketing strategiyasi yoz.
Byudjet: $200/oy. 
Format: 5 ta aniq qadam, har biri 2-3 jumla."
```
*Natija: Aniq, foydali, amaliy reja*

---

## Ushbu Modulda O'rganasiz:

### 1. Yaxshi prompt yozish qoidalari
- Aniqlik
- Struktura
- Misollar

### 2. Rol berish texnikasi
- "Sen ... sifatida ishla"
- Ekspert sifatida javob olish

### 3. Kontekst va cheklovlar
- Model nimani bilishi kerak
- Nimani qilmasligi kerak

### 4. Few-shot prompting
- Misollar bilan o'rgatish
- Pattern ko'rsatish

### 5. Chain-of-Thought
- Bosqichma-bosqich fikrlash
- Murakkab masalalar uchun

---

## Asosiy Formula

```
YAXSHI PROMPT = ROL + VAZIFA + KONTEKST + CHEKLOVLAR + FORMAT
```

### Misol:
```
ROL: "Sen tajribali huquqshunos advokatsansan."
VAZIFA: "Ish shartnomasi namunasi yoz."
KONTEKST: "O'zbekiston qonunchiligiga mos, IT kompaniya uchun."
CHEKLOVLAR: "Faqat asosiy bandlar, ortiqcha murakkablashtirma."
FORMAT: "10 ta band, har biri 1-2 jumla."
```
