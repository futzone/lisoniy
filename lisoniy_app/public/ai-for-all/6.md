# 2.2 Rol Berish Texnikasi

## "Sen ... sifatida ishla"

**Rol berish** — bu SIga ma'lum bir mutaxassis yoki shaxs sifatida javob berishni buyurish.

---

## Nima Uchun Ishlaydi?

SI milliardlab matnlarni o'qigan — jumladan, turli mutaxassislar yozgan matnlarni. Rol berganda, SI o'sha uslub va bilimlarni ishlatadi.

### Misollar:

**Shifokor roli:**
```
Sen tajribali oilaviy shifokorsan. Bemorga tushuntirganday, 
sodda tilda bosh og'rig'ining asosiy sabablarini tushuntir.
```

**Huquqshunos roli:**
```
Sen 15 yillik tajribaga ega O'zbekiston huquqshunosi.
Ish shartnomasi bo'yicha eng ko'p uchraydigan muammolarni tushuntir.
```

**Marketing mutaxassisi roli:**
```
Sen ijtimoiy tarmoqlar marketing ekspertisan.
Kichik biznes uchun Instagram strategiyasini tushuntir.
```

---

## Rol Berish Formulasi

```
Sen [kasb] san, [tajriba] bilan.
[Auditoriya] ga [vazifa] ni [uslub] da tushuntir.
```

### Misol:
```
Sen bosh oshpazsan, 20 yillik tajriba bilan.
Boshlovchilarga osh tayyorlashni oddiy va tushunarli tushuntir.
```

## Murakkab Rol Berish

Ba'zan ikki yoki undan ko'p rol berish foydali:

### Misol: Debat simulyatsiyasi
```
Sen ikkita shaxsni simulyatsiya qil:

SHAXS A: Tajribali investor, xavfdan qochuvchi
SHAXS B: Yosh tadbirkor, tavakkalchi

Mavzu: "Startapga investitsiya qilish kerakmi?"
Har biri 3 ta argument keltirsin.
```

### Misol: Qarama-qarshi fikr
```
Sen avval marketing menejer sifatida bu g'oyani himoya qil,
keyin moliyaviy direktor sifatida tanqid qil.
```

---

## Rol + Auditoriya

Rol berganda **auditoriyani** ham ayting:

```
Sen [ROL]san. 
[AUDITORIYA]ga [MAVZU]ni tushuntir.
```

### Misollar:

**Murakkab → Oddiy:**
```
Sen fizika professorisan.
10-sinf o'quvchisiga kvant mexanikasini oddiy tilda tushuntir.
```

**Texnik → Biznes:**
```
Sen IT arxitektisan.
Kompaniya direktoriga bulut texnologiyasining foydalarini 
texnik terminlarsiz tushuntir.
```
