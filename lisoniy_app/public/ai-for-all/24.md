# 5.4 Etik Masalalar va Copyright

## Mas'uliyatli SI Foydalanish

SI ishlatish oson ‚Äî lekin etik savollar ko'p.

---

## Asosiy Etik Savollar

### 1. Bu mening ishimmi?

```
Siz: SI bilan maqola yozdingiz
Savol: Bu sizning maqolangizmi?

Javob: Ha, AGAR:
- Siz g'oyani berdingiz
- Siz tahrir qildingiz
- Siz mas'uliyat olasiz
```

### 2. Aytish kerakmi?

```
Vaziyat: Hisobot yozishda SI yordam berdi
Savol: Aytish kerakmi?

Javob: Kontekstga bog'liq:
- Ichki hujjat ‚Üí Shart emas
- Ilmiy maqola ‚Üí Ha, albatta
- Mijozga ‚Üí Kompaniya siyosatiga qarang
```

### 3. Bu halolmi?

```
Vaziyat: Imtihon javobini SI yozdi
Javob: Yo'q ‚Äî bu aldash

Vaziyat: SI yordam bilan o'rgandingiz
Javob: Ha ‚Äî bu o'rganish
```
