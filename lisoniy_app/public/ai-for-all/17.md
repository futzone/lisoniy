# 4.2 Kundalik Vazifalarni Avtomatlashtirish

## Qaysi Vazifalarni Avtomatlashtirish Mumkin?

Har kuni takrorlanadigan, vaqt oladigan, lekin murakkab bo'lmagan vazifalar — avtomatizatsiya uchun ideal.

## ROI Hisoblash

```
Vaqt tejash = Haftalik soat × 4 hafta × Soatlik narx

Misol:
- Email saralash: 5 soat/hafta
- Soatlik narx: $20
- Oylik tejash: 5 × 4 × $20 = $400

Avtomatizatsiya xarajati: $50/oy
ROI: ($400 - $50) / $50 = 700%
```

## Oddiy Avtomatizatsiyalar (Zapier/Make)

### 1. Email → Task

```
Trigger: Yangi email (muhim label)
Action: Todoist/Asana da task yaratish
AI: Email dan task nomi va deadline ajratish
```

### 2. Form → Response

```
Trigger: Yangi form submission
Action: Email javob yuborish
AI: Shaxsiylashtirilgan javob yaratish
```

### 3. Calendar → Reminder

```
Trigger: Ertangi uchrashuv
Action: Telegram xabar
AI: Uchrashuv xulosa va tayyorgarlik ro'yxati
```
