# 2.5 Bosqichma-bosqich Fikrlash (Chain-of-Thought)

## SI ni "O'ylashga" Majburlash

**Chain-of-Thought (CoT)** — SI dan darhol javob o'rniga, bosqichma-bosqich fikrlashni so'rash.

---

## Nima Uchun Kerak?

Murakkab savollarda SI tez-tez xato qiladi. Lekin "bosqichma-bosqich o'yla" desangiz — aniqlik oshadi.

### Oddiy prompt:
```
Savol: "Agar 3 ta kishi 3 ta tortni 3 daqiqada pishirsa,
       9 ta kishi 9 ta tortni necha daqiqada pishiradi?"

SI: "9 daqiqa" (XATO!)
```

### Chain-of-Thought prompt:
```
Savol: "Agar 3 ta kishi 3 ta tortni 3 daqiqada pishirsa,
       9 ta kishi 9 ta tortni necha daqiqada pishiradi?"
       
Bosqichma-bosqich o'yla.

SI: 
1. 3 kishi 3 tortni 3 daqiqada pishiradi
2. Ya'ni har bir kishi 1 tortni 3 daqiqada pishiradi
3. 9 kishi bir vaqtda 9 tortni pishirishi mumkin
4. Har bir kishi 1 tort = 3 daqiqa
5. Javob: 3 daqiqa (TO'G'RI!)
```

---

## CoT ning Sehrli So'zlari

Quyidagi iboralarni qo'shing:

- "Bosqichma-bosqich o'yla"
- "Qadamma-qadam tushuntir"
- "Har bir qadamni ko'rsat"
- "O'z fikrlashingni tushuntir"
- "Let's think step by step" (inglizcha)

---

## Amaliy Misollar

### 1. Matematika

```
Savol: Do'konda 20% chegirma bor. Mahsulot narxi 150,000 so'm.
Qo'shimcha 10% cashback ham bor. Oxirida qancha to'layman?

Bosqichma-bosqich hisobla.

SI:
1. Asl narx: 150,000 so'm
2. 20% chegirma: 150,000 × 0.20 = 30,000 so'm
3. Chegirmadan keyin: 150,000 - 30,000 = 120,000 so'm
4. 10% cashback: 120,000 × 0.10 = 12,000 so'm qaytadi
5. Haqiqiy xarajat: 120,000 - 12,000 = 108,000 so'm
```

### 2. Qaror qabul qilish

```
Men yangi ish taklifini oldim. Qabul qilsam-mi?

Hozirgi ish: 5 mln so'm, uyga yaqin, barqaror
Yangi ish: 8 mln so'm, 1 soat yo'l, startup

Bosqichma-bosqich tahlil qil:
- Ijobiy tomonlar
- Salbiy tomonlar
- Risk baholash
- Tavsiya
```

### 3. Muammoni hal qilish

```
Muammo: Veb-saytimiz sekin ishlamoqda.

Bosqichma-bosqich diagnostika qil:
1. Mumkin bo'lgan sabablar
2. Har bir sababni qanday tekshirish
3. Eng ehtimoliy sabab
4. Yechim taklifi
```

## CoT Variatsiyalari

### 1. Zero-shot CoT
```
[Savol]
Bosqichma-bosqich o'yla.
```

### 2. Misollar bilan CoT
```
Misol:
Savol: 2+2*3=?
Fikrlash: Avval ko'paytirish: 2*3=6. Keyin qo'shish: 2+6=8.
Javob: 8

Sizning savol:
[Savol]
```

### 3. O'z-o'zini tekshirish
```
[Savol]
Bosqichma-bosqich o'yla.
Oxirida javobingni qayta tekshir.
```
