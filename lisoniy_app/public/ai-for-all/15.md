# 3.6 Jurnalistika va Kontent Yaratish

## SI — Yozuvchining Yordamchisi

Jurnalistlar va kontent yaratuvchilar uchun SI **tadqiqot, qoralama va tahrir**da yordam beradi.

---

## Asosiy Foydalanish

### 1. Maqola Yozish

```
[Mavzu] haqida maqola yoz:

Parametrlar:
- Uzunlik: [so'z]
- Uslub: [Yangilik / Tahlil / Feature / Opinion]
- Auditoriya: [kim]
- Ohang: [Rasmiy / Norasmiy / Academic]
- SEO: [kalit so'zlar]

Struktura:
- Headline (3 variant)
- Lead (kirish)
- Body (asosiy qism)
- Conclusion
```

### 2. Tahrir va Editing

```
Quyidagi matnni tahrir qil:

[Matn]

Tekshir:
- Grammatika va imlo
- Uslub izchilligi
- Aniqlik va lo'ndalik
- Oqim va o'tishlar

Ikki versiya ber: tahrirlangan + o'zgarishlar izohi
```

### 3. Tadqiqot Yordami

```
[Mavzu] bo'yicha tadqiqot qil:

Kerakli ma'lumotlar:
- Asosiy faktlar (5-7 ta)
- Statistika (so'nggi)
- Ekspert fikrlari
- Tarixiy kontekst
- Zid fikrlar

Format: Ro'yxat, qisqa, manba ko'rsatilgan
```

---

## Tayyor Promptlar

### Yangilik maqolasi:
```
[Voqea] haqida 500 so'zlik yangilik maqolasi.
5W1H (Kim, Nima, Qayerda, Qachon, Nima uchun, Qanday)
Neytral, faktlarga asoslangan.
3 ta headline variant.
```

### Blog post:
```
"O'zbekistonda IT sohasi" haqida 1500 so'zlik blog.
SEO optimized, kalit so'zlar: [...]
H2/H3 sarlavhalar bilan strukturalangan.
```

### Interview savollari:
```
[Shaxs] bilan [mavzu] bo'yicha intervyu uchun 15 ta savol.
Mix: Ice-breaker, main topic, follow-up, closing.
Ochiq savollar, qisqa javobli emas.
```

---

## Content Repurposing

```
Quyidagi blog postni boshqa formatlarga aylantir:

[Blog post]

Yaratilsin:
1. LinkedIn post (300 so'z)
2. Twitter thread (10 tweet)
3. Instagram carousel (10 slide, matn)
4. Email newsletter (500 so'z)
5. Podcast script (5 daqiqa)
```

---

## Plagiat va Etika

⚠️ **Muhim qoidalar:**

1. **SI yozgan = Sizning ismingiz bilan**
   - Mas'uliyat sizdа
   - Tekshirib chiqing

2. **Plagiat xavfi**
   - SI ba'zan mavjud matnlarni takrorlaydi
   - Plagiat checker ishlatishni odat qiling

3. **Faktlarni tekshiring**
   - SI "gallyutsinatsiya" qiladi
   - Muhim faktlarni mustaqil tekshiring

4. **O'z ovozingizni saqlang**
   - SI — qoralama
   - Yakuniy tahrir — siz

- ✅ Repurposing
