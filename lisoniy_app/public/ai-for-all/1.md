# 1.1 SI va Katta Til Modellari (KTM) Nima?

## Sun'iy Intellekt — Oddiy Tilda

**Sun'iy intellekt (SI)** — bu kompyuterlarni "aqlli" qiladigan texnologiyalar to'plami. Lekin bu "aql" inson aqliga o'xshamaydi.

### Oddiy tushuntirish:

Tasavvur qiling: siz bolaga minglab kitoblarni o'qib berganiz. Bola endi yangi jumlalar tuzishni, savolarga javob berishni o'rgangan — lekin u haqiqatan "tushunmaydi", faqat patternlarni (namunalarni) eslab qolgan.

**Katta Til Modellari (KTM)** xuddi shunday ishlaydi:
- Internetdagi milliardlab matnlarni "o'qigan"
- So'zlar orasidagi bog'lanishlarni o'rgangan
- Keyingi so'zni bashorat qiladi

---

## Asosiy Tushunchalar

### 1. Model
**Model** — bu matematik formula bo'lib, kirish (savol) asosida chiqish (javob) hosil qiladi.

```
Kirish: "Toshkent qaysi davlatda?"
↓ [Model]
Chiqish: "Toshkent O'zbekistonning poytaxti"
```

### 2. Token
**Token** — bu so'z yoki so'z qismi. Model tokenlar bilan ishlaydi.

```
"Salom, qalaysiz?" → ["Salom", ",", " qal", "ays", "iz", "?"]
= 6 token
```

### 3. Kontekst oynasi
**Kontekst oynasi** — model bir vaqtda "ko'ra oladigan" maksimal matn hajmi.

| Model | Kontekst |
|-------|----------|
| GPT-4 | ~128,000 token (~100 sahifa) |
| Claude 3 | ~200,000 token (~150 sahifa) |
| Gemini 1.5 | ~1,000,000 token (~750 sahifa) |

### 4. Parametrlar
**Parametrlar** — modelning "bilimi". Qancha ko'p bo'lsa, shuncha "aqlli".

| Model | Parametrlar |
|-------|-------------|
| GPT-4 | ~1.7 trillion |
| Claude 3 | Noma'lum |
| Llama 3 | 8B - 70B |

## SI Nima Qila OLADI?

✅ **Matn yozish** — maqola, email, hisobot
✅ **Savollarga javob** — tushuntirish, maslahat
✅ **Tarjima** — turli tillar orasida
✅ **Xulosa** — katta matnni qisqartirish
✅ **Kod yozish** — dasturlash yordam
✅ **Tahlil** — ma'lumotlarni ko'rib chiqish
✅ **G'oyalar** — brainstorming, kreativ fikrlar

---

## SI Nima Qila OLMAYDI?

❌ **Haqiqatni bilmaydi** — faqat o'rgangan narsani eslaydi
❌ **Internet emas** — real-time ma'lumot yo'q (maxsus vositlarsiz)
❌ **Matematik kafolat** — ba'zan xato hisoblaydi
❌ **So'nggi yangiliklar** — ma'lumotlar eskirgan bo'lishi mumkin
❌ **Hissiyot yo'q** — tushunganday ko'rinadi, lekin his qilmaydi

---

## Xulosa

KTM — bu juda kuchli **matn generatori**. U "tushunmaydi", lekin juda yaxshi "taqlid qiladi". Bu farqni tushunish — SI bilan samarali ishlashning kalitidir.
