# 4.3 Email va Xabarlar Bilan Ishlash

## Inbox Zero ‚Äî SI Bilan

Email ‚Äî ko'pchilik uchun eng katta vaqt yutuvchi. SI bu muammoni hal qilishga yordam beradi.

---

## Email Avtomatizatsiya Darajalari

### Daraja 1: Tasnif (Oddiy)
```
SI emailni kategoriyalaydi:
- Urgent (zudlik)
- Important (muhim)
- FYI (ma'lumot uchun)
- Archive (arxiv)
- Spam
```

### Daraja 2: Xulosa (O'rta)
```
SI uzun emailni qisqartiradi:
- 3 jumlada xulosa
- Asosiy so'rov
- Kerakli harakat
```

### Daraja 3: Javob Taklifi (Yuqori)
```
SI javob variantlarini tayyorlaydi:
- Qisqa tasdiqlash
- To'liq javob
- Boshqa vaqtga qo'yish
```

## Amaliy Sozlash

### Gmail + ChatGPT

**Qo'lda workflow:**
```
1. Emailni copy qiling
2. ChatGPT ga paste qiling
3. Prompt: "Bu emailni tahlil qil:
   - Muhimlik (1-5)
   - Xulosa (3 jumla)
   - Javob taklifi"
4. Natijani ko'rib chiqing
```

**Yarmi-avtomatik (Zapier):**
```
Trigger: Yangi email (label: "AI Process")
Action: ChatGPT ga yuborish
Action: Natijani Google Sheet ga yozish
Action: Telegram notification
```

## Messenger/Chat Avtomatizatsiya

### Telegram Bot

```
Foydalanish:
- Kunlik xulosa (scheduler bilan)
- Quick replies
- Task yaratish

Misol workflow:
1. Xabar keladi ‚Üí Bot tahlil qiladi
2. "Task" so'zi bor ‚Üí Todoist ga qo'shadi
3. Savol bor ‚Üí ChatGPT javob beradi
```

### Slack Integration

```
Slash command: /summarize

1. Kanal tarixini oladi (oxirgi 50 xabar)
2. ChatGPT ga yuboradi
3. Xulosa qaytaradi:
   - Asosiy muhokamalar
   - Qarorlar
   - Action items
```

## Email Shablonlari

### Avtomatik Javob Shabloni

```
Sen professional email yordamchisan.

Quyidagi emailga javob yoz:
[Email]

Qoidalar:
- Professional ohang
- 50-100 so'z
- Aniq va lo'nda
- Kerak bo'lsa savol so'ra
- Sign-off: "Best regards, [Ism]"
```

### Follow-up Shabloni

```
Quyidagi emailga follow-up yoz:

Original email: [...]
Yuborilgan: [sana]
Javob kelmagan: [necha kun]

Ohang: Do'stona eslatma, bosim yo'q
Uzunlik: 3-4 jumla
```
