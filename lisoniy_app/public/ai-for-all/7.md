# 2.3 Kontekst va Cheklovlar

## SI ga "Muhit" Yaratish

**Kontekst** — SI ga vazifani bajarish uchun kerakli ma'lumotlarni berish.
**Cheklovlar** — SI nima qilmasligi kerakligini aytish.

---

## Kontekst Nima Uchun Kerak?

SI sizning vaziyatingizni bilmaydi. Unga aytmasangiz — taxmin qiladi.

### Kontekstsiz:
```
Savol: "Taqdimot qil"
SI: [Umumiy, noaniq taqdimot]
```

### Kontekst bilan:
```
Savol: "Taqdimot qil.
Kontekst:
- Kompaniya: IT startup
- Auditoriya: Investorlar
- Vaqt: 10 daqiqa
- Maqsad: $500K investitsiya olish"

SI: [Aniq, maqsadli taqdimot]
```

---

## Kontekst Elementlari

### 1. Kim (Auditoriya)
```
- Investorlar uchun
- 5-sinf o'quvchilari uchun
- Texnik mutaxassislar uchun
```

### 2. Nima (Vazifa maqsadi)
```
- Sotish uchun
- O'rgatish uchun
- Tushuntirish uchun
```

### 3. Qayerda (Platforma/Muhit)
```
- LinkedIn posti uchun
- Email uchun
- Taqdimot uchun
```

### 4. Qachon (Vaqt/Muddat)
```
- 5 daqiqalik nutq
- 1 sahifalik hujjat
- Tez ko'rib chiqish uchun
```

---

## Cheklovlar (Constraints)

Cheklovlar — SI ni "yo'ldan chiqmaslik" uchun kerak.

### Format cheklovlari:
```
- 200 so'zdan oshmasin
- 5 ta punkt bo'lsin
- Jadval formatida
- Bullet pointlar bilan
```

### Uslub cheklovlari:
```
- Rasmiy tilda
- Jargonsiz
- Emoji ishlatma
- Sodda til
```

### Kontent cheklovlari:
```
- Faqat faktlar, fikr emas
- Manba ko'rsat
- Salbiy hech narsa yozma
- Faqat 2024-yil ma'lumotlari
```

## Misol: Marketing Post

```
# KONTEKST
- Kompaniya: "EcoMarket" — organik mahsulotlar do'koni
- Auditoriya: 25-40 yoshli ayollar, sog'lom turmush tarzi
- Maqsad: Yangi mahsulotni tanishtirish
- Platforma: Instagram

# CHEKLOVLAR
Format:
- 100-150 so'z
- Emoji ishlat (3-5 ta)
- Hashtag: 5-7 ta

Uslub:
- Do'stona, samimiy
- Sog'liqqa foydani ta'kidla
- Call-to-action qo'sh

Taqiqlar:
- Raqobatchilarni tilga olma
- Tibbiy da'volar qilma
- Narxni aytma
```

---

## Xulosa

**Kontekst** = SI ga "kim, nima, qayerda, qachon" ni aytish
**Cheklovlar** = SI ga "nima qilma, qanday bo'lmasin" ni aytish

Qancha aniq bo'lsa — javob shuncha yaxshi.
