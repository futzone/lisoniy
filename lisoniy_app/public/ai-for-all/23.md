# 5.3 Tekshirish va Tasdiqlash

## Fact-Checking Workflow

SI javobini tekshirish — oddiy lekin muhim ko'nikma.

---

## 3 Bosqichli Tekshirish

### Bosqich 1: Ajratish

SI javobidan faktlarni ajrating:

```
SI javobi:
"O'zbekiston aholisi 35 million (2024). 
Poytaxt Toshkent 2.5 million aholiga ega.
Prezident Shavkat Mirziyoyev 2016-yildan beri."

Faktlar:
□ Aholi: 35 million
□ Toshkent aholisi: 2.5 million  
□ Prezident: Shavkat Mirziyoyev
□ 2016-yildan beri
```

### Bosqich 2: Tekshirish

Har bir faktni tekshiring:

```
□ Aholi: 35 million
  → Google: "O'zbekiston aholisi 2024"
  → Natija: ~36 million ✓ (yaqin)

□ Toshkent aholisi: 2.5 million
  → Google: "Toshkent aholisi"
  → Natija: ~2.9 million ⚠️ (farq bor)

□ Prezident: Shavkat Mirziyoyev ✓
□ 2016-yildan: ✓
```

### Bosqich 3: Tuzatish

```
Tuzatilgan versiya:
"O'zbekiston aholisi ~36 million (2024). 
Poytaxt Toshkent ~3 million aholiga ega.
Prezident Shavkat Mirziyoyev 2016-yildan beri."
```

| Aniq raqamlar | Yuqori — har doim tekshiring |

| Tibbiy/huquqiy | Juda yuqori — mutaxassis kerak |

---

## Tez Tekshirish Texnikasi

### 1. "Eng muhim 3" qoidasi
```
Vaqt kam bo'lsa — eng muhim 3 faktni tekshiring.
Qolganlari muhim emas yoki oson tekshiriladi.
```

### 2. "Red flag" qidiruvi
```
Juda aniq raqamlar → tekshiring
Noma'lum odamlar/kitoblar → tekshiring
"Tadqiqotlar ko'rsatadiki..." → manba so'rang
```

### 3. Parallel tekshirish
```
SI yozayotganda — siz tekshiring.
Javob tugaganda — asosiy faktlar tayyor.
```

## Kompaniya uchun Verification SOP

```markdown
## SI Javoblarini Tekshirish Tartibi

### Har doim tekshiring:
- Raqamlar va statistika
- Ism va tashkilotlar
- Sanalar va voqealar
- Qonuniy ma'lumot

### Tekshirish ketma-ketligi:
1. SI javobini oling
2. Asosiy faktlarni belgilang
3. Har birini ishonchli manbada tekshiring
4. Noto'g'rilarini tuzating
5. Tuzatilgan versiyani ishlating

### Hujjatlashtirish:
- Tekshirilgan faktlar → yashil
- Tekshirilmagan → sariq
- Noto'g'ri topilgan → qizil + tuzatish
```
